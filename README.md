# Mastodon DMs

Prints out your Mastodon DMs on the command line.

## Usage

Set the MASTODON_ACCESS_TOKEN env variable in your profile first and then:

```
npm start
```

Or combine in the command:

```
MASTODON_ACCESS_TOKEN=[your access token] npm start
```
